Smart Irrigation avec ESP32 et Machine Learning 

Projet d’arrosage intelligent basé sur un ESP32 intégrant un modèle de Machine Learning embarqué pour prédire automatiquement la quantité d’eau nécessaire à l’irrigation.
Le système utilise des capteurs environnementaux (température, humidité du sol, luminosité, temps) et permet de réduire la consommation d’eau tout en automatisant l’arrosage.

Le modèle est entraîné en Python à partir de données réelles, puis converti en TensorFlow Lite et intégré directement dans l’ESP32.
En fonction des prédictions, l’ESP32 commande une pompe à eau via un relais et affiche les résultats sur le moniteur série.

Technologies clés : ESP32, Arduino/C++, Python, TensorFlow Lite, IoT, systèmes embarqués.

Perspectives : application mobile, gestion multi-plantes, historique des données et notifications intelligentes.
